激活函数

朴素感知机是指单层网络,指的是激活函数使用了阶跃函数的模型.
多层感知机是指神经网络,即使用sigmoid函数等平滑的激活函数的多层网络
感知机使用了阶跃函数作为激活函数,神经网络使用sigmoid函数或ReLU函数作为激活函数

sigmoid函数

sigmoid函数和阶跃函数的比较

不同: 感知机中神经元之间流动的是0或1的二元信号,而神经网络中流动的是连续的实数值信号.
相同: 阶跃函数和sigmoid函数两者都是非线性函数,sigmoid函数是一条曲线,阶跃函数是一条像阶梯一样的折线.

非线性函数

阶跃函数和sigmoid函数还有其他共同点，就是两者均为非线性函数。sigmoid函数是一条曲线，阶跃函数是一条像阶梯一样的折线，两者都属于非线性的函数。

在介绍激活函数时，经常会看到“非线性函数”和“线性函数”等术语。函数本来是输入某个值后会返回一个值的转换器。向这个转换器输入某个值后，输出值是输入值的常数倍的函数称为线性函数（用数学式表示为h(x)=cxh(x)=cxh(x)=cx。ccc为常数）。因此，线性函数是一条笔直的直线。而非线性函数，顾名思义，指的是不像线性函数那样呈现出一条直线的函数。

神经网络的激活函数必须使用非线性函数。换句话说，激活函数不能使用线性函数。为什么不能使用线性函数呢？
因为使用线性函数的话，加深神经网络的层数就没有意义了。
线性函数的问题在于，不管如何加深层数，总是存在与之等效的“无隐藏层的神经网络”。为了具体地（稍微直观地）理解这一点，我们来思考下面这个简单的例子。这里我们考虑把线性函数 h(x)=cxh(x) = cxh(x)=cx 作为激活函数，把y(x)=h(h(h(x)))y(x) = h(h(h(x)))y(x)=h(h(h(x)))的运算对应333层神经网络AAA。这个运算会进行y(x)=c×c×c×xy(x) = c \times c\times c\times xy(x)=c×c×c×x的乘法运算，但是同样的处理可以由y(x)=axy(x) = axy(x)=ax（注意，a=c3a = c^3a=c3）这一次乘法运算（即没有隐藏层的神经网络）来表示。如本例所示，使用线性函数时，无法发挥多层网络带来的优势。因此，为了发挥叠加层所带来的优势，激活函数必须使用非线性函数。

输出层的设计

神经网络可以用在分类问题和回归问题上,不过需要根据情况改变输出层的激活函数.一般而言,回归问题用恒等函数,分类问题用softmax函数.

恒等函数(回归问题)

恒等函数会将输入按原样输出,对于输入的信息,不加以任何改动地直接输出.

softmax函数(分类问题)

分类问题中使用的softmax函数可以用下面的式子表示

yk=exp⁡(ak+C)∑i=1nexp⁡(ai+C)\begin{aligned} y_k=\frac{\exp(a_k+C)}{\sum_{i=1}^n{\exp(a_i+C)}} \end{aligned}yk​=∑i=1n​exp(ai​+C)exp(ak​+C)​​

exp⁡(x)\exp(x)exp(x)是表示exe^xex的指数函数(e=2.7182...e=2.7182...e=2.7182...),式子中假设输出层共有nnn个神经元,计算第kkk个神经元的输出yky_kyk​.函数softmaxsoftmaxsoftmax的分子是输入信号aka_kak​的指数函数,分母是所有输入信号的指数函数的和.
为了防止溢出,我们在式子中加上常数CCC,一般在式子中C=−max(ai)C=-max(a_i)C=−max(ai​),这样能保证exp⁡(ak−max(ai))\exp(a_k-max(a_i))exp(ak​−max(ai​))不会溢出

问1: 解释一下为什么要判断x.T?
答1: 因为numpy的广播只能对最后一维进行,也就是按最后一维行进行广播,那么如果要做跨样本的广播(即每个样本选一个特征值,即列进行广播),那么就需要转置换成行,才能实现广播.
问2: 解释一下为什么要判断x.ndim==2?
答2: 因为数据最后在经过所有隐藏层后输出的数据应为(样本数N,one-hot-vector打分),比如MNIST数据集,输出类别应分为10类,那么one-hot-vector打分即为[0的分数,1的分数,...,9的分数],这里softmax的作用就是将分数进行normal(归一化)一下,使其值的范围落在(0.0,1.0)的区间.

需要注意的是: 即使使用了softmax函数,各个元素之间的大小关系也不会改变.这是因为指数函数(y=exp⁡(x)y=\exp(x)y=exp(x))是单调递增函数.即,aaa的各元素的大小关系和yyy的各元素的大小关系并没有改变.比如,aaa中最大的元素是第2个元素,yyy的最大值也是第2个元素.

  1. 一般而言,神经网络只把输出值最大的神经元所对应的类别作为识别的结果.
  2. 在实际问题中,由于指数函数的运算需要一定的计算机运算量,因此输出层的softmax函数一般会被省略.

输出层的神经元数量需要根据待解决的问题来决定.对于分类问题,输出层的神经元数量一般设定为类别的数量.
