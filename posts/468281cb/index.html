梯度和Hessians

 Recall that a matrix A∈Rn×n is symmetric if AT=A, that is, Aij=Aji for all i,j. Also  recall the gradient ∇f(x) of a function f:Rn→R, which is the n -vector of partial derivatives \begin{array}{l}{\text { Recall that a matrix } A \in \mathbb{R}^{n \times n} \text { is symmetric if } A^{T}=A, \text { that is, } A_{i j}=A_{j i} \text { for all } i, j . \text { Also }} \\ {\text { recall the gradient } \nabla f(x) \text { of a function } f : \mathbb{R}^{n} \rightarrow \mathbb{R}, \text { which is the } n \text { -vector of partial derivatives }}\end{array} Recall that a matrix A∈Rn×n is symmetric if AT=A, that is, Aij​=Aji​ for all i,j. Also  recall the gradient ∇f(x) of a function f:Rn→R, which is the n -vector of partial derivatives ​

∇f(x)=[∂∂x1f(x)⋮∂∂xnf(x)] where x=[x1⋮xn]\begin{aligned} \nabla f(x)=\left[\begin{array}{c}{\frac{\partial}{\partial x_{1}} f(x)} \\ {\vdots} \\ {\frac{\partial}{\partial x_{n}} f(x)}\end{array}\right] \text { where } x=\left[\begin{array}{c}{x_{1}} \\ {\vdots} \\ {x_{n}}\end{array}\right] \end{aligned}∇f(x)=⎣⎢⎡​∂x1​∂​f(x)⋮∂xn​∂​f(x)​⎦⎥⎤​ where x=⎣⎢⎡​x1​⋮xn​​⎦⎥⎤​​

 The hessian ∇2f(x) of a function f:Rn→R is the n×n symmetric matrix of twice partial  derivatives, \begin{array}{l}{\text { The hessian } \nabla^{2} f(x) \text { of a function } f : \mathbb{R}^{n} \rightarrow \mathbb{R} \text { is the } n \times n \text { symmetric matrix of twice partial }} \\ {\text { derivatives, }}\end{array} The hessian ∇2f(x) of a function f:Rn→R is the n×n symmetric matrix of twice partial  derivatives, ​

∇2f(x)=[∂2∂x12f(x)∂2∂x1∂x2f(x)⋯∂2∂x1∂xnf(x)∂2∂x2∂x1f(x)∂2∂x22f(x)⋯∂2∂x2∂xnf(x)⋮⋮⋱⋮∂2∂xn∂x1f(x)∂2∂xn∂x2f(x)⋯∂2∂xn2f(x)]\begin{aligned} \nabla^{2} f(x)=\left[\begin{array}{cccc}{\frac{\partial^{2}}{\partial x_{1}^{2}} f(x)} & {\frac{\partial^{2}}{\partial x_{1} \partial x_{2}} f(x)} & {\cdots} & {\frac{\partial^{2}}{\partial x_{1} \partial x_{n}} f(x)} \\ {\frac{\partial^{2}}{\partial x_{2} \partial x_{1}} f(x)} & {\frac{\partial^{2}}{\partial x_{2}^{2}} f(x)} & {\cdots} & {\frac{\partial^{2}}{\partial x_{2} \partial x_{n}} f(x)} \\ {\vdots} & {\vdots} & {\ddots} & {\vdots} \\ {\frac{\partial^{2}}{\partial x_{n} \partial x_{1}} f(x)} & {\frac{\partial^{2}}{\partial x_{n} \partial x_{2}} f(x)} & {\cdots} & {\frac{\partial^{2}}{\partial x_{n}^{2}} f(x)}\end{array}\right] \end{aligned}∇2f(x)=⎣⎢⎢⎢⎢⎢⎡​∂x12​∂2​f(x)∂x2​∂x1​∂2​f(x)⋮∂xn​∂x1​∂2​f(x)​∂x1​∂x2​∂2​f(x)∂x22​∂2​f(x)⋮∂xn​∂x2​∂2​f(x)​⋯⋯⋱⋯​∂x1​∂xn​∂2​f(x)∂x2​∂xn​∂2​f(x)⋮∂xn2​∂2​f(x)​⎦⎥⎥⎥⎥⎥⎤​​
