Numpy

np

  • array(object, dtype=None, copy=True, order=‘K’, subok=False, ndmin=0): 创建一个数组
  • dot(a,b,out=None): 两个数组点积
    • 如果a,b都为1维数组,直接向量的内积
    • 如果a,b都为$\ge$2维数组,使用矩阵乘法,但使用的是np.matmul或a@b(矩阵乘法)而不是np.multiply(a,b)或a*b(二者为元素级乘法)
      • 计算ab=∑i=0mab=a1b1+a2b2+⋯+ambm\mathbf {ab}=\sum_{i=0}^m{ab}=a_1b_1+a_2b_2+\cdots+a_mb_mab=∑i=0m​ab=a1​b1​+a2​b2​+⋯+am​bm​,可以分开为aaa和bbb,则可以直接写成np.dot(a,b)或a@b
      • 计算∑j=0n(θx−y)x\sum_{j=0}^n(\theta x-y)x∑j=0n​(θx−y)x,可以分开为(θx−y)(\theta x-y)(θx−y)和xxx,则可以直接写成np.dot(θx-y,x)或(θx-y)@x
      • 计算ab\mathbf {ab}ab(元素级)=[a1b1,a2b2,⋯ ,anbn][a_1b_1,a_2b_2,\cdots,a_nb_n][a1​b1​,a2​b2​,⋯,an​bn​],可以直接写成np.multiply(a,b)或a*b
      • 计算ai−aibi,i=0,1,2,⋯ ,na_i-a_ib_i,i=0,1,2,\cdots,nai​−ai​bi​,i=0,1,2,⋯,n,可以直接写成a-np.multiply(a,b)或a-a*b
    • 如果a或b为0维(也就是标量数值),等价于使用multiply,可以用np.multiply(a,b)或a*b(二者为元素级乘法)
    • 如果a为N维,b为1维数组,则是最后一个axis的a与b的积之和
  • full(shape, fill_value, dtype=None, order=‘C’): 返回一个给定shape和type,填充fill_value的数组
  • full_like(a, fill_value, dtype=None, order=‘K’, subok=True, shape=None): 返回一个与a相同shape且填充fill_value的数组
  • ones(shape, dtype=None, order=‘C’): 返回一个shape大小的全1数组
  • ones_like(a, dtype=None, order=‘K’, subok=True, shape=None): 返回一个与a相同shape大小的全1数组
  • row_stack: 将1D数组作为2D数组的行,等价于vstack
  • column_stack: 将1D数组作为2D数组的列等价于
    • 1D情况
    • 2D情况
  • vstack: 并列扩展,等价于row_stack
  • hstack: 并行扩展
  • r_: 并行连接
  • c_: 并列连接 r_,c_都可通过沿一个轴叠加数字来创建数组

np.random

  • rand(d0, d1, …, dn) 生成shape大小的随机数(0~1)

  • randn(d0, d1, …, dn) 生成shape大小的标准正态分布
    要表示正太分布 N(μ,σ2)N(\mu, \sigma^2)N(μ,σ2), 用下面形式:
    sigma * np.random.randn(...) + mu
    根据N(3, 6.25)随机生成2x4数组大小的样例:

  • randint(low[, high, size, dtype]) 从小(包含)到大(不包含)返回随机整数.

    • Parameters:
      • low : int
        Lowest (signed) integer to be drawn from the distribution (unless high=None, in which case this parameter is one above the highest such integer).
      • high : int, optional
        If provided, one above the largest (signed) integer to be drawn from the distribution (see above for behavior if high=None).
      • size : int or tuple of ints, optional
        Output shape. If the given shape is, e.g., (m, n, k), then m * n * k samples are drawn. Default is None, in which case a single value is returned.
      • dtype : dtype, optional
        Desired dtype of the result. All dtypes are determined by their name, i.e., ‘int64’, ‘int’, etc, so byteorder is not available and a specific precision may have different C types depending on the platform. The default value is ‘np.int’.
        New in version 1.11.0.
    • Returns:
      out : int or ndarray of ints
      size-shaped array of random integers from the appropriate distribution, or a single such random int if size not provided.
    • Examples

  • shuffle(x) 随机重排序列

    • Parameters:
      x : 要被重排的array或list
    • Examples 多维数组只会沿第一个轴进行重排

np.linspace

  • linspace(start, stop, num=50, endpoint=True, retstep=False, dtype=None, axis=0):
    按指定间隔值返回均匀的数
    返回num个均匀分布的数,间隔通过[start,stop]计算出
    间隔最后的点可以选择是否包含在内

np.matmul

  • matmul(): 等价于np.dot(a,b),a@b(数组的矩阵乘法)

np.matrix

  • Attributes
    • A: 返回ndarray(维数不变)
    • A1: 返回ndarray(一维)
    • H: 返回(复数)共轭转置[(complex)conjugate transpose]
    • I: 返回逆矩阵
    • T: 返回转置矩阵
    • flags: 内存布局
    • size: 返回总元素个数
  • getA(self)
    • 也可直接self.A,将自己作为ndarray返回,等价于np.asarry(self)

np.ndarray(为数组array的最原始类型)

  • argmax(axis=None, out=None): 返回沿给定轴的最大值的索引
  • argmin(axis=None, out=None): 返回沿给定轴的最小值的索引
  • min(axis=None, out=None, keepdims=False, initial=, where=True) 返回沿给定轴的最小值
    • Examples
      axis=0时,在列中选取最小值,axis=1时,在行中选取最小值
